Expected image files for Panorama Style website
================================================

Place all images in this directory (/public/images/).

Required files:
- hero.jpg           — Hero section background (full-screen, min 1920x1080, dark moody restaurant/event atmosphere)
- gallery-1.jpg      — Gallery image 1 (food, venue, events)
- gallery-2.jpg      — Gallery image 2
- gallery-3.jpg      — Gallery image 3
- gallery-4.jpg      — Gallery image 4
- gallery-5.jpg      — Gallery image 5
- gallery-6.jpg      — Gallery image 6
- gallery-7.jpg      — Gallery image 7
- gallery-8.jpg      — Gallery image 8
- gallery-9.jpg      — Gallery image 9
- venue-interior.jpg — Interior of the venue (used on gallery page)

Notes:
- If images are missing, GalleryGrid will show dark placeholder with camera icon
- Hero fallback: CSS gradient overlay will still look good on dark background
- Recommended size: 800–1600px wide, JPG quality ~80
